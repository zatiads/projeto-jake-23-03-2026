Quero criar uma skill para o Claude Code chamada "Fábrica de Criativos".

Contexto do projeto: é uma seção dentro de um sistema web pessoal chamado Jake OS (Flask + HTML/JS puro, banco Neon/PostgreSQL). O dono é um gestor de tráfego pago que cria conteúdo para seus clientes (Meta Ads, Google Ads).

O que a skill precisa saber fazer:

1. Geração de imagem via Replicate com seleção de modelo (Flux 1.1 Pro, Flux Dev, Recraft V3, Ideogram V3 Turbo, Imagen 4) — o usuário escolhe o modelo na hora

2. Engenharia de prompt automática com Claude antes de gerar — recebe um prompt simples e expande para algo profissional com iluminação, composição, estilo, etc.

3. Geração de vídeo via Replicate com seleção de modelo (Wan 2.2 5B Fast, Wan 2.2 I2V Fast, Wan 2.2 T2V Fast, Hailuo 02 Fast, Seedance 1 Lite, Runway Gen-4 Turbo) — cada modelo tem características diferentes (T2V, I2V, preço, qualidade)

4. Upload de imagem como input para vídeo (Image-to-Video)

5. Análise de referência — upload de imagem existente, Claude analisa o estilo e devolve um prompt para recriar aquele visual

Ainda está em fase de brainstorm. Preciso que você faça perguntas para entender melhor o escopo antes de escrever a skill.
