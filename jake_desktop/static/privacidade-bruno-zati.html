Política de Privacidade

Aplicativo: Bruno Zati | IA  ·  Última atualização: 15 de abril de 2026

Esta Política de Privacidade descreve como o aplicativo Bruno Zati | IA coleta, usa e protege as informações obtidas por meio da API do Meta (Facebook/Instagram) no contexto de gestão de tráfego pago e análise de campanhas publicitárias.

1. Informações Coletadas

O aplicativo pode acessar, por meio das permissões concedidas pelo usuário, as seguintes informações via API do Meta:

  • Dados de campanhas publicitárias (investimento, alcance, impressões, cliques, conversões)
  • Informações de contas de anúncios gerenciadas pelo usuário
  • Dados da Biblioteca de Anúncios (anúncios ativos públicos)
  • Métricas e relatórios de desempenho de anúncios

2. Uso das Informações

As informações coletadas são utilizadas exclusivamente para:

  • Análise de performance de campanhas de tráfego pago
  • Geração de relatórios e insights para gestão de anúncios
  • Pesquisa de referências criativas na Biblioteca de Anúncios do Meta
  • Otimização de campanhas publicitárias dos clientes

Nenhuma informação é vendida, repassada ou compartilhada com terceiros.

3. Armazenamento e Segurança

Os dados acessados via API do Meta são processados localmente em ambiente seguro e não são armazenados de forma permanente além do necessário para a geração dos relatórios solicitados. Adotamos medidas técnicas adequadas para proteger as informações contra acesso não autorizado.

4. Compartilhamento de Dados

Não compartilhamos dados obtidos via API do Meta com nenhuma empresa ou pessoa terceira. O acesso às informações é restrito ao gestor responsável pela conta.

5. Retenção de Dados

Os dados de campanhas são utilizados apenas durante a sessão de análise. Relatórios gerados são mantidos apenas pelo período necessário à prestação do serviço de gestão de tráfego.

6. Direitos do Usuário

O usuário pode, a qualquer momento:

  • Revogar o acesso do aplicativo às suas contas do Meta em Configurações do Facebook → Aplicativos
  • Solicitar informações sobre quais dados foram acessados
  • Solicitar a exclusão de qualquer dado armazenado

7. Uso da API do Meta

Este aplicativo utiliza a API do Meta em conformidade com os Termos de Serviço para Desenvolvedores do Meta e a Política de Dados do Meta. O uso é restrito às permissões explicitamente concedidas pelo usuário.
